◆ OF_LogLevel

enum OF_LogLevel

Debug log levels for Oficina.

Enumerator
OF_LOG_LVL_CRITICAL 

Critical output. Usually used on critical errors that WILL affect performance.

Warning
Critical output can be seen on console regardless of debug mode or not, unlike the others.
OF_LOG_LVL_ERROR 

Error output. Used in case of errors that will not affect performance too much.

OF_LOG_LVL_WARNING 

Warning output. Used in case of problems that will not affect performance, but may cause issues.

OF_LOG_LVL_INFO 

Information output. Used when outputting information that only has monitoring purposes.

OF_LOG_LVL_NONE 

Out-of-scope output. Will be printed regardless on the console, but only under Debug mode.
