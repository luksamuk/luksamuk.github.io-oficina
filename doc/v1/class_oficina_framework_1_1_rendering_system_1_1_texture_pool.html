static Font* OficinaFramework::RenderingSystem::TexturePool::LoadDefaultFont ( )
static

Loads the default font type.

Warning
As it does not produce a pointer to the used texture, the texture itself will be added to the TexturePool and will only be disposed when the application exits. If you're in extreme need for memory, use this wisely.
Returns
A pointer to the font to be used.
